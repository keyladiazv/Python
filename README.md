Actualmente estoy trabajando con contenido inspirado y complementado de los siguientes recursos:

## 🎯 Objetivo del repositorio

El propósito de este repositorio es:

- Practicar lógica de programación diariamente.
- Trabajar distintos tipos de datos, estructuras y algoritmos.
- Mantener evidencia pública y organizada de mi progreso en GitHub.
- Construir una base sólida para áreas futuras como:

✅ Automatización  
✅ Análisis de Datos  
✅ Ciberseguridad  
✅ Pentesting  
✅ Scripting ofensivo y defensivo  
✅ Desarrollo Backend
